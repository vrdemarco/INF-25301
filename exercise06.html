Learning to Use Classes and IDs

In this exercise, I added both an ID and a class to my HTML elements to show how they differ and how they help organize styles.

IDs are unique — used for one element only, while classes can be reused across multiple elements.

The layout also follows the CRAP design principles:

  • Contrast: dark header and nav against light content area.
  • Repetition: consistent color palette and spacing throughout the page.
  • Alignment: centered navigation and article for easy reading.
  • Proximity: related information grouped within sections for clarity.
